## Getting Started

1. **Open the application:**
   - Open `index.html` in your web browser

2. **Navigate sections:**
   - Use the header navigation to explore different sections
   - Home - Main introduction
   - Workouts - Weekly routine plans
   - Positions - Exercise gallery and form guides
   - BMI - BMI calculator
   - Diet - Nutrition planning
   - Cardio - Cardio workouts
   - Join - Contact section

3. **Track your progress:**
   - Monitor your daily workout completion
   - Use the BMI calculator for health metrics
   - Follow diet and nutrition guidelines
